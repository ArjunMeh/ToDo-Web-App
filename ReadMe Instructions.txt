Instructions for running the project
By Group 17,CS207 ADP 2022

Step 1: Copy the folder Our_TODO_G17 to C:\xampp\htdocs folder (or wherever the Xampp directory is located on your computer)
Step 2: Open the Xampp control panel and start MySQL
Step 3: Tap on Admin button for MySQL
Step 4: Go to import tab and click on Browse,to import the database sql file located within Our_TODO_G17
Step 5: Open a new browser tab and open the link localhost/Our_TODO_G17/Login.php
Step 6: The web application is now ready to use at your own convenience.
